# Security Policy

cloudterm is a client-side terminal UI component. It does not start shells, open network connections, or authenticate users by itself.

## Reporting a vulnerability

Please report security issues privately by emailing Jordan Coeyman at `jordan@coey.dev`.

Include:

- affected version or commit
- reproduction steps
- expected vs actual behavior
- any screenshots, traces, or proof-of-concept code that help explain the issue

I will acknowledge reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
